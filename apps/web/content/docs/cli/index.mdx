## char

Live transcription and audio tools

```
char [--base-url] [--api-key] [-m] [-l] [--record] [--no-color] [--base] [-v] [-q] [-h] [-V] <command>
```

**Global options:**

- `--base-url <BASE_URL>`
- `--api-key <API_KEY>`
- `-m, --model <MODEL>`
- `-l, --language <LANGUAGE>` (default: `en`)
- `--record`
- `--no-color`
- `--base <DIR>`
- `-v, --verbose` тАФ Increase logging verbosity
- `-q, --quiet` тАФ Decrease logging verbosity

**Options:**

- `-h, --help` тАФ Print help
- `-V, --version` тАФ Print version

**Subcommands:**

- `chat` тАФ Interactive chat with an LLM
- `listen` тАФ Start live transcription (TUI)
- `connect` тАФ Configure an STT or LLM provider
- `sessions` тАФ Browse past sessions
- `humans` тАФ Browse humans (contacts)
- `orgs` тАФ Browse organizations
- `status` тАФ Show configured providers and settings
- `auth` тАФ Authenticate with char.com
- `desktop` тАФ Open the desktop app or download page
- `bug` тАФ Report a bug on GitHub
- `hello` тАФ Open char.com
- `batch` тАФ Transcribe an audio file
- `model` тАФ Manage local models
- `completions` тАФ Generate shell completions

---

### char chat

Interactive chat with an LLM

```
char chat [--prompt] [--provider] [-h] [-V] <command>
```

**Options:**

- `--prompt <PROMPT>` тАФ Send a single prompt without entering the TUI (use `-` to read from stdin)
- `--provider <PROVIDER>`
- `-h, --help` тАФ Print help
- `-V, --version` тАФ Print version

**Subcommands:**

- `resume` тАФ Resume an existing chat session
