# General 

```bash
env-cmd --silent -f ./.env -- <COMMAND>
```

# Supabase

`task supabase-start` will generate the following environment variables:

```bash
SUPABASE_URL="http://127.0.0.1:54321"
VITE_SUPABASE_URL="http://127.0.0.1:54321"
SUPABASE_GRAPHQL_URL="http://127.0.0.1:54321/graphql/v1"
VITE_SUPABASE_GRAPHQL_URL="http://127.0.0.1:54321/graphql/v1"
SUPABASE_STORAGE_URL="http://127.0.0.1:54321/storage/v1/s3"
VITE_SUPABASE_STORAGE_URL="http://127.0.0.1:54321/storage/v1/s3"
DATABASE_URL="postgresql://postgres:postgres@127.0.0.1:54322/postgres"
SUPABASE_STUDIO_URL="http://127.0.0.1:54323"
SUPABASE_ANON_KEY="***REMOVED-LOCAL-SUPABASE-KEY***"
VITE_SUPABASE_ANON_KEY="***REMOVED-LOCAL-SUPABASE-KEY***"
SUPABASE_SERVICE_ROLE_KEY="***REMOVED-LOCAL-SUPABASE-KEY***"
S3_ACCESS_KEY="***REMOVED-LOCAL-S3-KEY***"
S3_SECRET_KEY="***REMOVED-LOCAL-S3-SECRET***"
S3_REGION="local"
```
