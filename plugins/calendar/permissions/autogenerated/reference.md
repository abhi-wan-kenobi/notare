## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-available-providers`
- `allow-is-provider-enabled`
- `allow-list-connection-ids`
- `allow-list-calendars`
- `allow-list-events`
- `allow-open-calendar`
- `allow-create-event`
- `allow-parse-meeting-link`
- `allow-google-account-status`
- `allow-google-import-client-json`
- `allow-google-import-client-file`
- `allow-google-connect`
- `allow-google-disconnect`
- `allow-google-reset`
