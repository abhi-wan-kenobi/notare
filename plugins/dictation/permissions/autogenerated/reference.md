## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-show`
- `allow-hide`
- `allow-set-phase`
- `allow-update-amplitude`
- `allow-show-orb`
- `allow-hide-orb`
- `allow-start-dictation`
- `allow-stop-dictation`
- `allow-is-dictating`
- `allow-type-text`
- `allow-deliver-text`
- `allow-clean-text`
