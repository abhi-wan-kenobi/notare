## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-get-settings`
- `allow-set-settings`
- `allow-set-secret`
- `allow-clear-secret`
- `allow-recent-deliveries`
- `allow-send-webhook`
- `allow-test-webhook`
